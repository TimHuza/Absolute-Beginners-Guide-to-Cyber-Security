# 7. Understanding Computer Networks & Communication

## What is a Network?
A network is a system of interconnected devices that can communicate with each other.

## Types of Networks
* **LAN** (Local Area Network): A network within a limited area (e.g., a home, school, or office).

* **WAN** (Wide Area Network): A larger network that spans multiple locations (e.g., the internet).

* **VPN** (Virtual Private Network): A secure way to connect to a network remotely.

* **Cloud Networks**: Services hosted in cloud environments rather than on-premises hardware.
