# 5. Difference Between Information Security and Cyber Security

**`Information Security`** is the broader field that covers all types of protection for all types of data

**`Cyber Security`** is more like a specialized field under information security


### Example on SSD
If we will take the SSD out of computer, the only way to access to take it physically and put it in another computer. That's the **`Information Security`**.

**`Cyber Security`** is when the SSD is connected to the computer and we are protecting it from hackers. SO digitaly we are protecting the data.
